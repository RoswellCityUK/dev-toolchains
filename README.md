# dev-toolchains

A mono-repo of reusable development toolchain Docker images (devcontainers + CI-ready).

## Images
- `toolchain-latex` — TeX Live + latexmk + biber, intended for VS Code LaTeX Workshop devcontainers.
- `toolchain-quarto` — Quarto + TeX Live for Amazon KDP publishing.

## Publishing
Images are published to GHCR:
- `ghcr.io/roswellcityuk/dev-toolchains/toolchain-latex`

See `images/latex/README.md` for usage.
